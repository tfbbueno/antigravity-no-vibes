
# Sobre

Um repositório dedicado a armazenar skills customizadas para o **Antigravity**. O objetivo aqui é automatizar processos e garantir que a IA ajude a aplicar boas práticas de desenvolvimento de verdade, e não só *vibe-codar*.


## Instalação: Onde colocar as skills?

Para que o Antigravity reconheça essas automações localmente, os diretórios de cada skill (contendo o arquivo `SKILL.md`) devem ser copiados para a pasta de configuração do Antigravity no seu computador.

**Caminho padrão no Windows:**
```bash
C:\Users\<SeuUsuario>\.gemini\antigravity\skills\

```

📄 [Leia mais na documentação](https://codelabs.developers.google.com/getting-started-with-antigravity-skills?hl=pt-br#4)

> **Dica:** Após adicionar ou editar uma skill nessa pasta, digite o comando `/skills reload` ou `/skills refresh` diretamente no chat do Antigravity para forçar a reindexação sem precisar reiniciar.
